# Project Overview

## High-Level Architecture

The high-level architecture of the THBLD project consists of multiple components that work together to achieve the project's goals. Below is a description of the key components:

### 1. Frontend
- **Technology Stack**: React, Redux
- **Description**: The user interface of the application, allowing users to interact with the system.

### 2. Backend
- **Technology Stack**: Node.js, Express
- **Description**: The server-side logic that handles requests, manages data, and communicates with the database.

### 3. Database
- **Technology Stack**: MongoDB
- **Description**: A NoSQL database used to store application data and user information.

### 4. API
- **Description**: RESTful APIs to connect the frontend with the backend, facilitating data exchange and user interaction.

### 5. Deployment
- **Platforms**: Heroku, AWS
- **Description**: Cloud services used for deploying and managing the application.

### 6. CI/CD
- **Description**: Continuous Integration and Continuous Deployment processes to ensure code quality and streamline deployment.
