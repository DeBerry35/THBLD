# InstaTrack Project Documentation

## Overview
InstaTrack is a social media analytics tool designed to help users track and analyze their interactions on Instagram.

## Features
- Track follower growth over time
- Analyze engagement metrics (likes, comments, shares)
- Visualize performance trends
- Export analytics in various formats

## Installation
1. Clone the repository: `git clone https://github.com/DeBerry35/THBLD`
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Usage
- Login with your Instagram credentials to start tracking your performance.
- Navigate through the dashboard to view analytics and insights.
